# INSTITUTIONAL PRICE ACTION PATTERN BREAKDOWN
## Complete Analysis of Cheat Sheet Patterns

---

## 1. QML (QUASIMODO LEVEL) SETUPS
**What it is:** A failed swing high or low that creates an "order block" — institutional resting orders.

**Visual Pattern:**
```
    /\
   /  \     ← Previous high (failed)
  /    \
 /      \
/________\___  ← QML Level (order block)
          \
           \____  ← Price returns to QML for entry
```

**Why it works:**
- Institutions left unfilled orders at the failed high/low
- When price returns, these orders get hit
- Creates high-probability reversal zone
- 75% reliability on H1-H4 timeframes

**Trade Setup:**
- Entry: Return to QML level
- Stop: Below/above QML + buffer
- Target: Opposite swing or 2:1 RR

---

## 2. SUPPLY/DEMAND FLIPS
**What it is:** Old resistance becomes support (bullish flip) or old support becomes resistance (bearish flip)

**Visual Pattern - Bullish Flip:**
```
    /\      ← Resistance
   /  \
  /    \____
 /          \
/____________\____  ← Old resistance now support
              /\
             /  \   ← Bounces from flipped zone
```

**Why it works:**
- Institutions who sold at resistance now buy at same level
- Psychology flip from "too expensive" to "good value"
- Previous sellers become buyers
- 70% reliability

**Trade Setup:**
- Wait for price to return to flipped zone
- Look for rejection candle (pin bar, engulfing)
- Entry on confirmation
- Stop below/above zone

---

## 3. FAKEOUTS (BREAKOUT TRAPS)
**What it is:** Price briefly breaks above resistance or below support, triggers stops, then reverses hard

**Visual Pattern:**
```
         /\  ← Fake breakout
        /  \
       /    \___
      /         \
_____/           \___  ← Traps breakout buyers
    |    /\    |
    |___/  \___|       ← Returns to range
```

**Why it works:**
- Institutions need liquidity to exit positions
- They push price past level to trigger retail stop orders
- Absorb the liquidity, then reverse
- Classic "stop hunt"
- 68% reliability

**Trade Setup:**
- Wait for break + immediate rejection
- Entry on return below/above level
- Tight stop (the fakeout range)
- Target: opposite side of range or 2:1 RR

---

## 4. LIQUIDITY GRABS
**What it is:** Intentional sweep of highs/lows to trigger stop orders before reversal

**Visual Pattern:**
```
    /\  ← Equal highs (liquidity pool)
   /  \
  /    \  /\
 /      \/  \  ← Sweeps highs
/_______/    \___  ← Reverses hard
```

**Why it works:**
- Equal highs = clusters of stop orders
- Institutions push price to sweep the stops
- Creates liquidity for their reversal position
- Price reverses immediately after sweep
- 72% reliability

**Trade Setup:**
- Identify equal highs/lows (liquidity pools)
- Wait for sweep + rejection
- Entry on reversal candle
- Stop beyond sweep + buffer

---

## 5. COMPRESSION INTO EXPANSION
**What it is:** Low volatility coil (consolidation) building energy before explosive move

**Visual Pattern:**
```
    /\\
   /  \\
  /    \\
 /      \\
/        \\____  ← Compression (lower highs, higher lows)
\        /
 \      /
  \    /
   \  /
    \/  ← EXPLOSION (breakout)
```

**Why it works:**
- Energy builds during consolidation
- Eventually breaks with momentum
- Range expansion = volatility expansion
- Measured move = height of consolidation pattern
- 65% reliability (direction less certain than magnitude)

**Trade Setup:**
- Wait for breakout of compression zone
- Entry on confirmed break (close outside)
- Stop inside compression zone
- Target: Projection of consolidation height

---

## 6. STOP HUNTS THAT LOOK LIKE BREAKOUTS
**What it is:** Brief violation of level to trigger retail stops, appears as breakout but immediately reverses

**Visual Pattern:**
```
       /\  ← Looks like breakout
      /  \
_____/    \____
     \    /
      \  /  ← Immediate reversal
       \/
```

**Why it works:**
- Institutions need your stop loss for liquidity
- Push price just past your stop
- Your loss is their liquidity
- Price immediately returns
- 70% reliability

**Trade Setup:**
- Watch for break + immediate failure
- Entry on reversal candle
- Tight stop (the hunt range)
- Target: opposite side

---

## 7. FLAG LIMITS (ORDER BLOCKS)
**What it is:** Price returns to a specific order block (institutional zone) for entry

**Visual Pattern:**
```
Strong move up:
    /|
   / |
  /  |
 /   |
/____|____  ← Flag limit/Order block
      |
      |  ← Price returns to block
      |___  ← Entry here
```

**Why it works:**
- Institutions build positions in order blocks
- Price returns to fill remaining orders
- High-probability continuation entry
- 73% reliability

**Trade Setup:**
- Identify strong move + consolidation
- Mark the order block (last bullish/bearish candle before move)
- Wait for return to block
- Entry on rejection from block

```
 /

```

**Head and Shoulders:**
```
    /\
   /  \
  /    \    /\    /\
 /      \  /  \  /  \
/________\/    \/    \___
       (head) ( shoulders)
```

**Why they work:**
- These patterns exist because institutions use them
- Self-fulfilling at key zones
- Market structure levels
- 74% reliability at key zones

**Trade Setup:**
- Wait for pattern completion
- Entry on neckline break (H&S) or second rejection (double top)
- Stop beyond pattern extreme
- Target: Pattern height projection

## PATTERN RECOGNITION LOOP (Pre-Trade)

**Step 1: Identify Pattern**
- Which pattern is present? (QML, Flip, Grab, etc.)
- Is it clean or messy?
- What timeframe? (H1, H4, D1)

**Step 2: Map Liquidity**
- Where are the stops sitting?
- Where is the liquidity pool?
- What would institutions want to do here?

**Step 3: Check Sentiment**
- Does sentiment confirm pattern direction?
- Pattern says bullish, sentiment >60? ✅
- Pattern says bullish, sentiment 50? ⏳ WAIT

**Step 4: Context Filter**
- Gamma extreme? Adjust size/stops
- Political alert? Confirm direction
- News catalyst? Check alignment

**Step 5: Execute or Wait**
- ALL checks pass? → Execute via Alpaca
- Missing something? → Wait for confirmation
- Grade too low? → Skip trade

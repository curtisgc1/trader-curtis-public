# Core Strategy - Liquidity Patterns

## 8 Patterns Ranked by Reliability
1. QML (75%) — Failed swing = order block. Price returns to fill institutional orders.
2. Institutional Reversal (74%) — Double tops, H&S at key zones. Institutions use these.
3. Flag Limit (73%) — Return to order block for continuation entry.
4. Liquidity Grab (72%) — Sweep equal highs/lows to harvest stops, then reverse.
5. Supply/Demand Flip (70%) — Old resistance becomes support. Psychology flip.
6. Stop Hunt (70%) — Brief violation triggers your stop, then reverses.
7. Fakeout (68%) — False breakout traps retail.
8. Compression → Expansion (65%) — Low vol coil builds energy before explosive move.

## Core Principle
Price moves to liquidity. Every pattern exists to push price where orders are sitting so institutions can harvest retail stops.
